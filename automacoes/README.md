# Automações N8N — Heroic Leap Health

Coloque os arquivos `.json` exportados do N8N aqui.

## Como exportar do N8N:
1. Abra o workflow
2. Menu `...` → **Export**
3. Salve o arquivo `.json` nesta pasta

## Workflows esperados:
- `secretaria.json` — Orquestrador principal (fluxo de conversa)
- `agente-conhecimento.json` — Consulta base de informações
- `agente-agendamento.json` — Roteador de agenda
- `executor-agenda.json` — Executor por profissional
- `crm.json` — Atualização do banco após cada interação
- `followup-1.json` — Follow up nível 1
- `followup-2.json` — Follow up nível 2
- `followup-3.json` — Follow up nível 3
